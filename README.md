### Frontend

1. Navigate to the frontend directory:

2. clone repository:

    ```sh
    cd https://github.com/tracie96/todoFE.git
    ```

    ```sh
    cd todofe
    ```

2. Install dependencies:

    ```sh
    yarn install
    ```

3. Start the React app:

    ```sh
    yarn start
    ```

    The frontend will be running on `http://localhost:3000` or `http://localhost:3000`.
